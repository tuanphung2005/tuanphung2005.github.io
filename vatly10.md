## Công thức vật lí thường dùng (với tôi)

# Ném ngang

Xác định chuyển động
```
Thời gian: t = sqrt(2h/g)
Tầm ném xa: L = x(max) = v0.t= v0.sqrt(2h/g)
Dạng quỹ đạo: y = g/(2.v0^2).x^2
```

Chuyển động thành phần
```
Trục Ox:
  ax = 0
  vx = v0
  x = v0.t
```
```
Trục Oy
  ay = g
  vy = g.t
  y = 1/2.g.t^2
```
